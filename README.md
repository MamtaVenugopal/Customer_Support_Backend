# Ensemble dev-test metrics (fixed thresholds)

All documentation and paths below are **inside `best_model_code/`** only.

This file summarizes **confusion matrices**, **AUC-ROC**, **F1**, **F0.5** (Fβ with β=0.5), and **F2** for four rank-average ensembles on the **dev-test** split (600 clips). Metrics are computed on **each subset**: overall, sections `00–02`, and domains `source` / `target`.

**Also:** [Pooled max-Fβ thresholds (median-of-distinct-optima)](#median-fbeta-threshold-tables) — full tables for **18 phase-2 checkpoint models** plus **C+mix, B+C+mix, A+B+C+mix, A+B+C** (overall + section 00–02), with TN/FP/FN/TP, AUC-ROC, AUC-PR, Precision, Recall, F0.5–F3.

## Method

- **Score**: rank-average of per-model reconstruction scores (same construction as `sweep_threshold_multi_combo.py` / `sweep_threshold_cmix_devtest.py`).
- **Label**: binary anomaly = `label == anomaly` (otherwise normal).
- **Decision**: `predict_anomaly = (score >= threshold)`.
- **AUC-ROC**: `sklearn.metrics.roc_auc_score(y_true, score)` on that subset — i.e. **continuous ensemble score vs binary anomaly label**. Reported as **nan** only if a subset has a single class (does not occur here).

Machine-readable copy of all rows: `ensemble_fixed_threshold_section_report.csv`.

## Pooled max-Fβ thresholds, median-of-distinct-optima, dev-test (600 clips)

For each model, β ∈ {0.5, 1, 2, 3}, the threshold **maximizes Fβ** on **unique** reconstruction/rank scores (`select_threshold_max_fbeta`). **median_thr** = median of **distinct** Fβ-optimal thresholds; **min_FP+FN_thr** minimizes FP+FN on observed scores in [span_min, span_max]. Tables below use **median_thr**.
